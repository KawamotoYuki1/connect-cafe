## ポイント制度
- 毎月1日に **+1,000pt** が自動で追加されます
- ポイントは **繰り越し** されます（月末に消失しません）
- ポイントでの購入は **1日1品まで** です
- 2品目以降はPayPayでお支払いください

## 購入方法

### ポイントで購入する場合
1. メニューから商品をタップ
2. 下部の「ポイント消費」ボタンをタップ
3. 「購入を確定」をタップ → 完了

### PayPayで購入する場合
1. メニューから商品をタップ
2. 下部の「PayPay」ボタンをタップ
3. 「購入を確定」をタップ
4. 購入記録が保存されます
5. **カメラが起動するので、カフェに設置のPayPay QRコードを読み取ってください**
6. **PayPayアプリで表示された金額を手入力して支払いを完了してください**

⚠️ **PayPayの支払いはご自身で完了してください。** アプリ上で購入記録は自動で残りますが、実際の支払いはPayPay QRコードから行う必要があります。
